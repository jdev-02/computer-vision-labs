# Computer Vision Labs

Jupyter notebook progression through computer vision techniques — from pixel-level operations to real-time object detection and tracking with YOLO.

## Contents

| Notebook | Topic |
|----------|-------|
| `pixel-operations.ipynb` | Pixel manipulation, color channels, histograms |
| `spatial-filtering.ipynb` | Convolution, edge detection, Gaussian/Sobel filters |
| `kmeans-example.ipynb` | K-means clustering for image segmentation |
| `classification-1.ipynb` | Image classification fundamentals |
| `classification-torch.ipynb` | Classification with PyTorch |
| `CNN.ipynb` | Convolutional neural network architecture and training |
| `tracking.ipynb` | Object tracking fundamentals |
| `ultralytics_yolo_lab_with_semantic_capture_quiz.ipynb` | YOLO object detection with semantic scene capture |
| `yolo_tracking_benchmark_notebook_v3_with_semantic_quiz.ipynb` | Multi-object tracking benchmark with YOLO |

## Progression

```
Pixel Operations → Spatial Filtering → K-Means Segmentation
       ↓
Classification (sklearn) → Classification (PyTorch) → CNN Training
       ↓
Object Detection (YOLOv8) → Object Tracking Benchmark
```
